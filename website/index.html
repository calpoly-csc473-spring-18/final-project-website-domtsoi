Scene Animation Using Simplex Noise Function and Reflections

Dominic Tsoi - CSC 473 - Spring 2018 - Ian Dunn

Project Description

My Name is Dominic Tsoi, for my Advanced Rendering final Project I chose to implement a simplex noise sampling function created by GitHub User tombsar. I also implemented a rendering loop that allowed for multiple scene rendering. Lastly I stitched together the rendered images with blender's video editing software.

Additionally in the scene I kept the implemented reflection code that I have from early in ther quarter. I also increment the color values over time, as well as apply different color patterns depending on the shapes.
